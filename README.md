Endless Blood is a fast-paced roguelite survival game inspired by the gameplay mechanics of Vampire Survivors and set in a dark world aesthetically reminiscent of Bloodborne.
The game focuses on intense combat, constant leveling up, and regularly rewarding the player with new items and upgrades. Players face increasingly difficult waves of enemies while 
numerous visual and sound effects, combined with gradual character progression, create an engaging gameplay loop designed to keep players immersed and encourage repeated runs.
